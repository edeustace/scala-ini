Help

The list of puzzles is on the left. The highlighted puzzle contents are underneath this popup in a code editor. To solve the puzzle you just have to fill in all the '?' symbols, so that any evaluations within the puzzle evaluate to true.
So if you have something like: 

true == ?
replace ? with 'true' so that you have: 
true == true
Because this evaluation will return true - you will have completed the puzzle successfully.

Ctrl + R will run the test.
