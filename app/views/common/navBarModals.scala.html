About

Scala Puzzles is a puzzle site for the programming language Scala.
It is inspired by 4clojure.com but its functionality has evolved in a different direction. It is meant to allow users to build up their knowledge through the medium of puzzles.

Currently there is a list of puzzles that start easy and become more difficult.
You can also create your own puzzles to share with the world.

Please send feedback to ed[dot]eustace at google's mail thingy.

Help

The list of puzzles is on the left. The highlighted puzzle contents are underneath this popup in a code editor. To solve the puzzle you just have to fill in all the '?' symbols, so that any evaluations within the puzzle evaluate to true.
So if you have something like: 

true == ?
replace ? with 'true' so that you have: 
true == true
Because this evaluation will return true - you will have completed the puzzle successfully.

Ctrl + R will run the test.
Sign up and we'll keep track of your progress.
